## Topic: Void Functions, Parameters, and Arguments

Today I learned about **void return values**.

A void function is a function that does not return a value back to the caller.

Example:

```cpp
#include <iostream>

// void means the function does not return a value
void printHi()
{
    std::cout << "Hi" << '\n';
}

int main()
{
    printHi();

    return 0;
}
```

In this example, the function performs an action by printing text to the screen, but it does not return any information back to the program.

This showed me that not every function needs to calculate and return a value. Some functions simply perform a task.

### Application to Rise of Nations

Void functions could be useful for game systems that only need to display information.

Examples:

```cpp
displayResources();
displayCountryInfo();
showMainMenu();
displayDiplomacyOptions();
```

These functions may not need to return anything because their primary purpose is displaying information to the player.

## Function Parameters and Arguments

I also learned about parameters and arguments.

### Parameters

A parameter is a variable declared in a function's header.

Parameters allow a function to receive information from the caller.

Example:

```cpp
void printValues(int x, int y)
```

The variables `x` and `y` are parameters.

### Arguments

Arguments are the actual values passed into a function when it is called.

Example:

```cpp
printValues(6, 7);
```

The values `6` and `7` are arguments.

### Pass By Value

I learned that when a function is called, the arguments are copied into the function's parameters.

This process is called **pass by value**.

Example:

```cpp
#include <iostream>

void printValues(int x, int y)
{
    std::cout << x << '\n';
    std::cout << y << '\n';
}

int main()
{
    printValues(6, 7);

    return 0;
}
```

When the function is called:

```cpp
printValues(6, 7);
```

the values `6` and `7` are copied into the parameters `x` and `y`.

The function then prints those values.

## Applying Parameters to a Calculator

I realized that parameters and arguments can be used to build a simple calculator.

The function can receive two numbers, perform a calculation, and return the result.

For example:

```cpp
int add(int a, int b)
{
    return a + b;
}
```

The parameters `a` and `b` receive values from the caller, and the function returns their sum.

## Calculator Project

To practice functions, I created a basic calculator program.

```cpp
#include <iostream>

int add(int a, int b)
{
    return a + b;
}

int subtract(int a, int b)
{
    return a - b;
}

int multiply(int a, int b)
{
    return a * b;
}

int divide(int a, int b)
{
    if (b != 0)
        return a / b;
    else
    {
        std::cout << "Error: Division by zero is not allowed." << '\n';
        return 0;
    }
}

int main()
{
    int num1, num2;

    std::cout << "Enter two integers: ";
    std::cin >> num1 >> num2;

    std::cout << "The sum of "
              << num1 << " and "
              << num2 << " is: "
              << add(num1, num2) << '\n';

    std::cout << "The difference of "
              << num1 << " and "
              << num2 << " is: "
              << subtract(num1, num2) << '\n';

    std::cout << "The product of "
              << num1 << " and "
              << num2 << " is: "
              << multiply(num1, num2) << '\n';

    std::cout << "The quotient of "
              << num1 << " and "
              << num2 << " is: "
              << divide(num1, num2) << '\n';

    return 0;
}
```

This program combines several concepts I have learned so far:

* Variables
* User input
* Return values
* Void functions
* Parameters
* Arguments
* Mathematical operators
* Conditional logic

I also added a simple check to prevent division by zero.
