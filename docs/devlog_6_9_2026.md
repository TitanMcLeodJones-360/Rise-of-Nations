## Topic: Output, Input, and Basic Menu Systems

Today I learned more about output in C++ using `std::cout`. I explored ways to improve the readability of program output by using newline characters (`\n`) and `std::endl`.

Examples:

```cpp
std::cout << "Hello\n";
std::cout << "World" << std::endl;
```

Both methods move the output to a new line, helping organize information displayed to the user. This will be useful in future projects because clear formatting makes programs easier to read and navigate.

### User Input

I continued learning about user input using `std::cin`. I created a simple country selection menu:

```cpp
#include <iostream>

int main()
{
    std::cout << "Please Select a Country:\n";
    std::cout << "1. United States\n";
    std::cout << "2. Canada\n";
    std::cout << "3. Mexico\n";

    int x{};
    std::cin >> x;

    std::cout << "You have selected: " << x << std::endl;

    return 0;
}
```

This program displays a list of countries, accepts a choice from the user, and then displays the selected option.

### Connection to Rise of Nations

This lesson directly relates to the development of *Rise of Nations*. The game will require players to make many decisions, including:

* Selecting a country
* Choosing diplomatic actions
* Managing resources
* Trading with other nations
* Making military decisions

The menu system I created today is a simple version of how players may interact with the game in the future.

### Understanding Operators

I also learned the difference between the stream operators `<<` and `>>`.

```cpp
std::cout << x;
```

The `<<` operator sends data to an output stream, displaying information on the screen.

```cpp
std::cin >> x;
```

The `>>` operator extracts data from an input stream and stores it in a variable.

Understanding these operators is important because they allow programs to communicate with users.

### Reflection

Today's lesson helped me better understand how programs display information and receive user input. While the country selection menu is simple, it represents one of the first interactive systems that could eventually become part of *Rise of Nations*. I am beginning to see how individual C++ concepts can be combined to create larger game mechanics.
