## Topic: Literals and Operators

Today I learned about literals and operators in C++.

### Literals

A literal is a fixed value written directly into source code. Examples include:

```cpp
5
10
3.14
'A'
"Hello"
```

Literals themselves cannot be changed because they represent specific values.

I learned the difference between a literal and a variable through the following example:

```cpp
#include <iostream>

int main()
{
    std::cout << 5 << '\n';

    int x { 5 };
    std::cout << x << '\n';

    return 0;
}
```

The first line prints the literal value `5`.

The second example creates a variable named `x` and initializes it with the literal value `5`. Unlike the literal itself, the variable can later be changed by the program.

### Resource Simulation

I applied this concept by creating a small oil consumption simulation.

```cpp
#include <iostream>

int main()
{
    int oil { 5 };
    int oilUsage {};

    std::cout << "The Amount of Oil you have is: "
              << oil << " gallons\n";

    std::cout << "How much oil would you like to consume?\n";

    std::cin >> oilUsage;

    std::cout << "You have consumed "
              << oilUsage << " gallons of oil\n";

    std::cout << "You now have: "
              << oil - oilUsage
              << " gallons of oil\n";
}
```

This program starts with 5 gallons of oil, asks the user how much oil they wish to consume, and then calculates the remaining amount.

One issue I noticed is that the program allows the user to consume more oil than is available, resulting in negative values.

This is a problem I would like to solve in the future as I learn more about C++. However, negative values may still have useful applications in Rise of Nations, such as representing:

* National debt
* Budget deficits
* Economic decline

### Operators

I also learned about operators.

Operators perform actions on values and variables.

Example:

```cpp
std::cout << 1 + 2 << '\n';
```

In this example:

* `1` and `2` are operands
* `+` is the operator

The result is the value `3`, which is printed to the console.

Common operators include:

```cpp
+
-
*
/
=
```

I also learned that some operators are represented by keywords, such as:

```cpp
new
delete
throw
```

These perform special operations within the language.

### Operator Vocabulary

#### Unary Operators

Unary operators act on one operand.

Example:

```cpp
-5
```

The unary minus operator takes the operand `5` and changes its sign to `-5`.

#### Binary Operators

Binary operators act on two operands.

Example:

```cpp
34 + 20
```

The `+` operator combines the left operand (`34`) and right operand (`20`) to produce the value `54`.

Other examples of binary operators include:

```cpp
<<
>>
```

These are the stream insertion and extraction operators used with `std::cout` and `std::cin`.

#### Ternary Operators

Ternary operators act on three operands.

I was introduced to this concept today and plan to learn more about it in the future.

#### Nullary Operators

Nullary operators act on zero operands.

I also plan to study these more in depth later.

### Chaining Operators

I learned that operators can be chained together to perform more complex calculations.

Example:

```cpp
2 * 3 + 4
```

The multiplication operation is performed first:

```cpp
2 * 3 = 6
```

Then the addition operation is performed:

```cpp
6 + 4 = 10
```

This follows the rules of operator precedence, which are similar to the order of operations used in mathematics (PEMDAS).

This concept will be especially useful for my future C++ Multi-Use Calculator project, where calculations may involve multiple operations in a single expression.
